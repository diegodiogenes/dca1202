Detailed Description

A classe Poligono define a estrutura e funcionalidades de um polígono, contituindo-se como um conjunto de pontos que, quando manipulado corretamente, é possível realizar-se operações de calculo de área, transladar o polígono ou até mesmo rotacioná-lo.
